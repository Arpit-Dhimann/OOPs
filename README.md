# Object-Oriented Programming (OOPs) in Java

Welcome to my **OOPs** repository! This repository is a collection of my learning journey in Java, specifically focusing on Object-Oriented Programming concepts. Here, I implement core principles and features of Java to build robust and maintainable code.

## 🚀 Repository Overview
This repository contains practical examples and documentation explaining how to design programs using classes, objects, and other OOP paradigms.

## 📂 Contents
Each file in this repository represents a core OOP concept:

| File Name | Concept |
| :--- | :--- |
| `BasicClassAndObject.java` | Classes & Objects fundamentals |
| `MethodsInClass.java` | Defining and using methods |
| `UsingThisKeyword.java` | Understanding the `this` keyword |
| `NonParameterizedConstructor.java` | Implementing default constructors |
| `ParameterizedConstructor.java` | Using constructors with parameters |
| `CopyConstructor.java` | Object initialization via copying |
| `NOTES.md` | Comprehensive theory and notes on OOPs |

## 🛠️ Concepts Covered
*   **Class & Objects**: Logic and runtime entity representation.
*   **Constructors**: Non-Parameterized, Parameterized, and Copy constructors.
*   **`this` Keyword**: Referencing current class instances.
*   **Memory Management**: Heap vs Stack allocation in Java.
*   **Garbage Collection**: Automatic memory deallocation.

## 📖 Documentation
You can find the detailed theory and comprehensive notes for all the above topics in the `NOTES.md` file or the exported `OOPS_Concepts_Java.pdf` document.
